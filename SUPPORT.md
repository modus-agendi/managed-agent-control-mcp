# Support

Need help using **managed-agent-control-mcp**?

- 📖 Start with the [README](README.md) and the [`docs/`](docs/) guides
  (configuration, authentication, deployment, architecture, tools).
- 💬 Ask a question in
  [GitHub Discussions](https://github.com/modus-agendi/managed-agent-control-mcp/discussions).
- 🐛 Found a bug or want a feature? Open an
  [issue](https://github.com/modus-agendi/managed-agent-control-mcp/issues)
  using the templates.
- 🔒 Security problem? Follow [SECURITY.md](SECURITY.md) — **do not** open a
  public issue.

Please don't include secrets (API keys, tokens) in issues or discussions.
